📱 App de Navegação

📌 Descrição do Projeto
Este projeto é um aplicativo Android desenvolvido utilizando Jetpack Compose.  
O objetivo é demonstrar a implementação de navegação entre telas utilizando o componente Navigation Compose.

O app possui múltiplas telas e permite ao usuário navegar entre elas de forma simples e intuitiva.

---

🚀 Tecnologias Utilizadas
- Kotlin
- Jetpack Compose
- Navigation Compose
- Material 3

---

📱 Funcionalidades
- Navegação entre telas
- Interface moderna com Compose
- Uso de componentes reutilizáveis
